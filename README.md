# 🚀 冲刺备考引擎 (StudyEngine)

一款基于 Python + Flet 构建的极简、沉浸式桌面端备考效率工具。专为高强度冲刺阶段设计，提供番茄钟与正向计时双引擎，支持本地数据安全存储与全维度复盘统计。

## ✨ 核心特性

- **⏳ 双核计时引擎**：支持“🧱 筑城 (正向计时)”与“🌱 种树 (番茄倒计时)”无缝切换。
- **🌲 沉浸式图鉴**：每一次专注都会化作你的专属森林，支持日/周/月多维度回溯。
- **📊 雷达级统计**：自动计算各科目时间投入占比，进度条直观呈现偏科情况。
- **🛡️ 坚如磐石的数据安全**：纯本地 JSON 存储，支持一键备份导出，内置防误触与崩溃日志自救机制。
- **🌙 跨天无缝结转**：内置“凌晨 2 点”逻辑分割线，深夜修仙学习的数据自动精准归档至昨日。

## 🤖 自动化部署 (CI/CD) 说明

本项目已接入 **GitHub Actions**。当你将代码推送到 `main` 或 `master` 分支时，云端服务器将自动为你打包最新的 Windows `.exe` 可执行文件。

### 📦 如何获取最新版的 .exe 软件？

1. 点击仓库页面顶部的 **[Actions]** 标签页。
2. 在左侧栏点击 **Build Desktop Executable** 工作流。
3. 点击列表中最上方（最新）的一次运行记录（带有绿色 ✔️ 标记）。
4. 滑动到页面最下方找到 **Artifacts** 区域。
5. 点击下载 **`StudyEngine-Windows`** 压缩包。
6. 解压后即可直接双击运行 `.exe` 文件，无需安装任何 Python 环境！
